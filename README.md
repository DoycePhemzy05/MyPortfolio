# DoycePhemzy Graphixs — Portfolio Website

A personal portfolio website for **DoycePhemzy Graphixs** — a graphic designer and frontend developer blending visual storytelling with technical craft.

---

## 🌐 Live Demo

> Coming soon — [https://doycephemzygraphixs.vercel.app/]

---

## ✨ Features

- Fully responsive design (mobile, tablet, desktop)
- Smooth scroll reveal animations
- Active nav highlighting on scroll
- Project showcase with hover effects
- Skills & tools section
- About / values section
- Contact section with live social links

---

## 🛠️ Built With

| Technology | Purpose |
|---|---|
| HTML5 | Structure & content |
| CSS3 | Styling, layout, animations |
| Vanilla JavaScript | Scroll effects, interactivity |
| Google Fonts | Playfair Display + DM Sans |

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/DoycePhemzy05/portfolio.git
   ```
2. Open the project folder
3. Double-click `index.html` to open in your browser — no server needed!
